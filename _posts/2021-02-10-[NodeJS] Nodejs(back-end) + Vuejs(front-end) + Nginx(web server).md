## Install NodeJS
### nvm
```shell
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
$ nvm --version
```

### npm
```shell
$ nvm ls-remote
$ nvm install --lts
$ nvm ls
$ node --version
$ npm --version
```
